# Hotel Booking Dashboard

An interactive Hotel Booking Dashboard project created using Power BI and Excel to analyze hotel booking trends, customer behavior, cancellations, revenue insights, and occupancy performance.

---

## Project Overview

This dashboard helps hotel management understand:

- Booking trends
- Customer behavior
- Revenue performance
- Occupancy analysis
- Cancellation insights
- Seasonal booking patterns

The project focuses on transforming raw hotel booking data into meaningful business insights through interactive visualizations.

---

## Tools & Technologies Used

- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
- Business Analytics

---

## Dataset Details

The dataset contains hotel booking information including:

- Hotel Type
- Booking Date
- Arrival Month
- Lead Time
- Customer Country
- Market Segment
- Room Type
- Booking Status
- ADR (Average Daily Rate)
- Number of Guests
- Reservation Details

1. Data Collection

---

## Sample Business Insights

- City hotels experience higher cancellation rates.
- Peak bookings occur during holiday seasons.
- Online bookings generate maximum reservations.
- Repeat guests contribute to stable occupancy rates.
